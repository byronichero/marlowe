Reports

Report generation

Generate reports by assessment, framework, or date range. The API endpoint GET /api/v1/reports accepts optional filters: assessment_id, framework_id, from_date, to_date.

Report content and AI-assisted summaries can be added in a future release. For now, use the API or Dashboard for overview data.
