Help

Getting started

Marlowe helps you manage AI governance, responsible AI, privacy, and global regulations in a framework-agnostic way. You can work with any standard (e.g. EU AI Act, GDPR, NIST AI RMF) without vendor lock-in.

  • Use the Dashboard for an overview and quick links.
  • Add frameworks and requirements via the API or future admin UI; then browse them under Controls.
  • Create assessments and attach evidence to requirements.
  • Upload documents; they are parsed and can be searched and used in AI Chat.
  • Explore the Knowledge graph to see how frameworks, requirements, and assessments relate.
Key features

Frameworks & requirements — CRUD for frameworks (e.g. EU AI Act, GDPR) and their requirements; filter by level or family.

Assessments — Track status and link to frameworks and organizations.

Evidence — Attach files to requirements and assessments; stored in object storage.

Documents — Upload PDF, DOCX, etc.; content is extracted and indexed for search and AI.

AI Chat — Ask questions; answers can use your documents and framework data.

Knowledge graph — Visualize relationships between controls, requirements, and frameworks.

API

The backend exposes a REST API under /api/v1. OpenAPI docs are at /docs when the backend runs on port 4000.
